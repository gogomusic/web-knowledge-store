| 包名   | 作用         | 安装              | 常用命令/示例                                                |
| ------ | ------------ | ----------------- | ------------------------------------------------------------ |
| nvm    | node版本管理 | `npm i nvm -g`    | `nvm list`查看已安装的node版本<br />`nvm list available`查看所有可以安装的node<br />`nvm install <版本号>`安装指定版本node<br />`nvm use <版本号>`使用指定版本node |
| rimraf | 快速删除文件 | `npm i rimraf -g` | `rimraf node_modules`                                        |
